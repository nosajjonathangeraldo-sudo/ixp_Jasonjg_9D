Kinematics

Definition

Kinematics is the study of how things move. It explains speed, distance, and direction without talking about forces.

Distance and Displacement

Distance: Total path length traveled.

Displacement: Shortest straight-line distance with direction.

Speed and Velocity

Speed: Rate at which an object covers distance.

Speed = Distance / Time

Velocity: Rate at which an object changes position in a direction.

Velocity = Displacement / Time

Acceleration

Rate at which velocity changes over time.

Acceleration = Change in Velocity / Time

Average Speed

Total distance divided by total time.

Average Speed = Total Distance / Total Time

Distance-Time Graph

    In a distance-time graph, the slope shows speed.
    - Straight line: constant speed
    - Curved line: changing speed
    - Horizontal line: object not moving
    - Steep slope: faster motion
